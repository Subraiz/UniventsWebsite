What Is Univents?

An Easier Way To Market Events


Univents is a centralized event planning application for students
at any school across the country. We take a more personalized 
approach when it comes to creating/hosting events and attending 
events. We focus on displaying events that the user wants to see, and
not being bombarded with information. We aim to solve a lot of issues
that both student clubs and administrative orginizations face when 
it comes to increasing awareness.
